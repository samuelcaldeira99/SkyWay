# ✈ SkyWay Airlines — Protótipo Android

Aplicativo mobile desenvolvido como protótipo funcional para teste de usabilidade,
na disciplina de Processos de Software (IHC) — Univates.

## 📱 Telas

- **Tela 1 — Pesquisa de Viagem**: busca de voos com origem, destino, datas, passageiros e classe
- **Tela 2 — Seleção de Assentos**: grade interativa com seleção de até 3 assentos

## 🎯 Objetivo

Aplicar um teste de usabilidade com usuário real, utilizando a técnica Think Aloud,
conforme especificado na Unidade 5 da disciplina.
